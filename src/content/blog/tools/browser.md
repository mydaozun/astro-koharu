# github开源免费指纹浏览器

## XChrome
- GitHub: https://github.com/chanawudi/XChrome
- 官网: https://web3tool.vip/
- 免费开源，只支持 Windows
- 支持全功能指纹隔离 + 群控 + 脚本运行 + API
- 37.5MB

## GeekezBrowser
- GitHub: https://github.com/EchoHS/GeekezBrowser
- 官网：https://browser.geekez.net/
- 一款基于xray-core代理的开源指纹浏览器
- 380MB
![geeezbrowser主界面](https://raw.githubusercontent.com/EchoHS/GeekezBrowser/main/docs/Main%20Interface1.png)


## undetectable-fingerprint-browser
- GitHub: https://github.com/itbrowser-net/undetectable-fingerprint-browser
- 英文项目，定位是 Multilogin/Incogniton/Kameleo 的开源平替
- 高度可定制，支持 Canvas/WebGL/User-Agent 等 spoofing
- 自动化友好（Selenium/Playwright/Puppeteer）
- 适合：爬虫、自动化、多账号脚本跑的人
- 234 MB
